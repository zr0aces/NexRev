# คู่มือการใช้งาน NexRev

ยินดีต้อนรับสู่ NexRev! คู่มือนี้จะช่วยให้คุณเริ่มต้นจัดการ Pipeline การขายของคุณได้อย่างมีประสิทธิภาพ

---

## 1. การเริ่มต้นใช้งาน
- **การเข้าสู่ระบบ**: ใช้ชื่อผู้ใช้และรหัสผ่านของคุณ (ค่าเริ่มต้นคือ `admin` / `admin`)
- **การแถบเมนู**: ใช้แถบด้านบนเพื่อสลับระหว่างหน้า **Today** (วันนี้), **Pipeline** (รายการขาย), และ **Activity Log** (ประวัติกิจกรรม)

## 2. โปรไฟล์และการแจ้งเตือน Telegram
- **แถบ Profile**: ไปที่แถบ **Profile** เพื่อจัดการบัญชีของคุณ
- **ความปลอดภัย**: คุณสามารถเปลี่ยนรหัสผ่านได้ในส่วน Security
- **การตั้งค่า Telegram**: คลิก **🔗 Link Automatically** เพื่อเชื่อมต่อกับ Telegram ทันทีโดยไม่ต้องใส่ ID เอง (หรือใส่ Chat ID เองหากต้องการ)
- **สรุปรายวัน**: ทุกเช้าเวลา **8:30 น.** คุณจะได้รับสรุปรายการที่ต้องติดตามส่งตรงถึง Telegram

## 3. การจัดการงานรายวัน (หน้า Today)
- **แดชบอร์ดรายวัน**: ดูรายการติดตามผลที่เร่งด่วนในรูปแบบการ์ด 3 คอลัมน์
- **ตัวบ่งชี้ความเร่งด่วน**: 
    - 🔴 **สีแดง**: เลยกำหนดติดตามผล
    - 🟡 **สีส้ม**: ครบกำหนดวันนี้ หรือภายใน 3 วัน
- **ทางลัด**: คลิกที่การ์ดเพื่อไปยังหน้ารายละเอียดของลูกค้านั้นๆ ในหน้า Pipeline ทันที

## 3. การจัดการโอกาสการขาย (หน้า Pipeline)
- **การค้นหา**: ใช้แถบค้นหาเพื่อหาชื่อลูกค้าหรือคีย์เวิร์ด
- **การกรองตามขั้นตอน**: คลิกที่ป้ายสถานะ (เช่น Prospecting, Negotiation) เพื่อกรองรายการ
- **นโยบาย 1 รายการต่อ 1 ลูกค้า**: ลูกค้าแต่ละรายจะมีโอกาสการขายได้เพียงรายการเดียวเท่านั้น หากคุณค้นหาชื่อลูกค้าที่มีอยู่แล้วในขณะเพิ่มรายการใหม่ ระบบจะพาคุณไปยังรายการเดิมทันที

## 4. เครื่องมืออัตโนมัติและ AI
- **การบันทึกโน้ต**: ใส่รายละเอียดในช่องบันทึกแล้วคลิก **Log Note**
- **การดึงงานอัตโนมัติ**: คลิก **📋 Extract tasks** เพื่อให้ AI อ่านโน้ตของคุณและสร้างรายการงาน (Kanban) ให้โดยอัตโนมัติ
- **การสรุปส่ง Salesforce**: คลิก **▪ SF update note** เพื่อรับสรุปกิจกรรมที่พร้อมคัดลอกไปวางใน Salesforce (โดยสรุปเฉพาะกิจกรรมใหม่ที่เกิดขึ้นหลังจากครั้งล่าสุดเท่านั้น)
- **การสรุปด้วย AI**: ใช้ **✨ AI summarize** เพื่อเปลี่ยนโน้ตการประชุมที่ยุ่งเหยิงให้เป็นหัวข้อที่อ่านง่ายและเป็นมืออาชีพ

## 5. กระดาน Kanban
- **To Do**: งานที่ต้องเริ่มทำ
- **Follow-ups**: รายการที่อยู่ระหว่างรอการตอบกลับ
- **Done**: งานที่เสร็จสมบูรณ์แล้ว
- **การลากและวาง**: คุณสามารถลากการ์ดสลับไปมาในแต่ละคอลัมน์เพื่ออัปเดตสถานะได้ทันที

## 6. เคล็ดลับเพื่อความสำเร็จ
- **ตั้งวันติดตามผลเสมอ**: เพื่อให้มั่นใจว่ารายชื่อลูกค้าจะปรากฏในหน้า **Today** ของคุณ
- **สำรองข้อมูลสม่ำเสมอ**: กดปุ่ม **Backup** ที่เมนูด้านบนสัปดาห์ละครั้ง
- **ความเป็นส่วนตัว**: ข้อมูลของคุณจะถูกเก็บไว้ในเซิร์ฟเวอร์ของคุณเอง ไม่มีการส่งข้อมูลไปยังคลาวด์ภายนอก
