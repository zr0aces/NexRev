คู่มือการใช้งาน NexRev Master Guide

แหล่งข้อมูลหลักสำหรับการจัดการ Pipeline การขาย, กิจวัตรประจำวัน และการใช้งาน AI

1. การเริ่มต้นใช้งาน

  • การเข้าสู่ระบบ: ใช้ชื่อผู้ใช้และรหัสผ่าน (ค่าเริ่มต้น: admin / admin)
  • การใช้งานบนคอมพิวเตอร์: สลับระหว่างหน้า Today, Pipeline, Activity Log และ Profile
  • การใช้งานบนมือถือ: ใช้ Bottom Nav และ ปุ่มลอย (+) เพื่อความรวดเร็ว

2. กิจวัตรประจำวันของคุณ

  • วางแผน (เช้า): ตรวจสอบแดชบอร์ด Today รายการสีแดงคือเลยกำหนด สีส้มคือใกล้ครบกำหนด
  • ดำเนินการ: บันทึกโน้ตการประชุมลงในส่วนกิจกรรมทันที บันทึกแบบดิบๆ แล้วให้ AI ช่วยจัดระเบียบภายหลัง
  • อัตโนมัติ: ใช้ปุ่ม 📋 Extract tasks เพื่อสร้างรายการงานใน Kanban จากโน้ตของคุณทันที
  • ซิงค์ข้อมูล: สร้าง ▪ SF update notes เพื่อสรุปกิจกรรมสำหรับอัปเดต Salesforce ได้อย่างรวดเร็ว

3. โปรไฟล์และการแจ้งเตือน Telegram

  • การตั้งค่า Telegram: เชื่อมต่อบัญชีในแถบ Profile เพื่อรับสรุปรายวันเวลา 8:30 น.
  • ความปลอดภัย: เปลี่ยนรหัสผ่านทันทีเพื่อให้ข้อมูล Markdown ส่วนตัวของคุณปลอดภัย

4. Pipeline และ Kanban

  • การค้นหา: ค้นหาบัญชีลูกค้าได้ทันที โดยยึดหลัก 1 ลูกค้า = 1 โอกาสการขาย
  • Kanban: ติดตามงานย่อยในรูปแบบ To Do → Follow-ups → Done
  • มุมมองบนมือถือ: แตะชื่อบัญชีเพื่อดูรายละเอียด แตะ "Back" เพื่อกลับไปที่รายการ

5. เครื่องมือเพิ่มประสิทธิภาพ AI

  • ✨ AI Summarize: เปลี่ยนโน้ตที่ยุ่งเหยิงให้เป็นหัวข้อสรุปที่เป็นมืออาชีพ
  • 📋 Task Extraction: สแกนบันทึกเพื่อหาหัวข้องานและสร้างการ์ด Kanban ให้โดยอัตโนมัติ

6. คีย์ลัด (สำหรับคอมพิวเตอร์)

  • Ctrl + K : เพิ่มโอกาสการขายใหม่
  • Esc : ปิดหน้าต่าง / ยกเลิกการค้นหา
  • Enter : เพิ่มรายการใน Kanban อย่างรวดเร็ว

7. เคล็ดลับเพื่อความสำเร็จ

  • จัดระเบียบหน้า Today: อัปเดตวันติดตามผลเสมอเพื่อให้หน้า Today แสดงเฉพาะงานปัจจุบัน
  • Markdown-First: ข้อมูลของคุณถูกเก็บในรูปแบบ Markdown ที่อ่านง่าย การสำรองข้อมูลทำได้ง่ายๆ เพียงคัดลอกโฟลเดอร์ data/
